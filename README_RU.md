# NVRIER - Терминальный Тир-Лист 🔥

[Read in English (Читать на английском)](README.md)

Интерактивная полноэкранная утилита с текстовым интерфейсом (TUI) для составления тир-листов прямо внутри терминала Kitty. Рендеринг изображений происходит в 100% оригинальном пиксельном качестве с использованием аппаратного протокола Kitty Graphics Protocol.

# !ВНИМАНИЕ!

Этот проект на половину составлен ИИ, могут быть баги. Если вы увидите баги, составьте issue.

## Возможности

* **Оригинальное качество пикселей**: Аппаратный рендеринг изображений с ускорением на GPU через Kitty Protocol.
* **Интерактивный TUI**: Работа в выделенном полноэкранном буфере на движке `termbox-go`. Управление происходит мгновенно, без необходимости нажимать кнопку `Enter`.
* **Локализация**: Быстрое переключение языка интерфейса между русским и английским на лету.
* **Сохранение настроек**: Утилита автоматически запоминает выбранный вами язык и применяет его при следующем запуске.

## Требования

1. **Терминал**: Для работы утилиты строго необходим **терминал Kitty** или любой другой эмулятор терминала, полностью поддерживающий Kitty Graphics Protocol.
2. **Окружение Go**: В системе должен быть установлен компилятор Go версии 1.16 или выше.

## Установка

Скачайте репозиторий и запустите интерактивный скрипт установки из корневой папки проекта:

```bash
git clone https://github.com/neverloseagain1/nvrtier

cd nvrtier

chmod +x install.sh

./install.sh
```

В начале установки выберите `1` для русского языка. Скрипт автоматически скачает зависимости, скомпилирует исходный код, создаст целевую папку для картинок и переместит бинарный файл в системную директорию `/usr/local/bin/`.

## Использование

1. Скопируйте нужные изображения (`.png`, `.jpg` или `.jpeg`) в созданную папку в вашей домашней директории:
   ```bash
   ~/nvrtier_pictures
   ```
2. Запустите программу из любого места в терминале:
   ```bash
   nvrtier
   ```

## Управление

* `Стрелка влево` / `Стрелка вправо`: Листать текущий склад доступных изображений.
* `1` - `5`: Отправить выбранную картинку в соответствующую строку тира.
* `L`: Переключить язык интерфейса (Русский ↔ Английский).
* `Q` / `Ctrl+C`: Очистить графические слои терминала и выйти из программы.
